props 옵션 Vue.component('porps-simple',{ props:['msg','hello'] }); el 옵션 el옵션은 Vue 인스턴스가 동작할 DOM 엘리먼트를 지정할때 사용합니다.el은 Vue 템플릿을 컴파일 하는 시점에 Vue 컴포넌트가 생성한 DOM으로 대체되므로 html 엘리먼트와 body 엘리먼트를 Vue인스턴스의 el 옵션값으로 설정하는 것은 권하지 않습니다. 사용예) template 옵션 template 옵션은 Vue 인스턴스의 마크업으로 사용할 문자열을 지정합니다. script 태그를 사용해 템플릿 블록 읽어 오기 render 옵션 문자열을 전달하는 템플릿 대신 자바스크립트의 프로그래밍 기능을 활용할때 사용합니다. 컴포넌트 옵션(el,render,template)중 우선순위가 높습니다. 사용예) Vue.component('render-acro',{ render:function(h){ return h( 'h1',this.$slot.default ) } }); parent 옵션 선언한 컴포넌트 의 상위 Vue 컴포넌트를 설정하는데 사용합니다. 사용예) var parent_vm= new Vue({}); var child_vm = new Vue({ parent:parent_vm }); mixins 옵션 mixins 옵션값은 객체 리터럴로 설정하며 객제 리터럴은 vue 인스턴스 옵션 및 뒤에서 설명하게 될 라이프사이클 이벤트 훅을 담아둔다. 사용예) var mixin = { created:function(){console.log(1);}, data:{a:'hello'} }; var vm = Vue({ created:function(){console.log(2)}, mixins:[mixin] data:{a:'hello2'} }) name 옵션 옵션값으로 문자열 형태만 허용합니다 사용예) var vm = new Vue({ name:'Vue VM ROOT COMPONENT', el:'div#hello' });
